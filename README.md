# Weather app

This project is a weather app built with HTML, CSS, and JavaScript. The program uses a API to give real-time data of any city the user decides to search up. 

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

To run this project, you'll need the following:

A modern web browser (e.g., Chrome, Firefox, Edge).

A text editor or IDE (e.g., Visual Studio Code) if you want to modify the code.

### Installing

Clone the repository or download the ZIP file:

git clone https://github.com/Aryangoel225/Weather-app  

Navigate to the project folder:

cd Weather-app

Open the index.html file in your browser to view the application:

Right-click on index.html and select "Open with" > Your Browser.

## Running the Application

Once the index.html file is open in your browser, you can interact with the program, by input the city you wish to search up. The weather information––temperature, humidity, cloud coverage, and an emoji representing the information. 

### Break down into end to end tests

This project is simple and does not include automated tests. You can manually verify its functionality by interacting with the buttons in the browser.


## Built With

* HTML: Structure of the web page.
* CSS: Styling for the application.
* JavaScript: Logic for button functionality.
